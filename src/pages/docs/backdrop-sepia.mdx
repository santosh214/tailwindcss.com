## Usage

Use the `backdrop-sepia` and `backdrop-sepia-0` utilities alongside the `backdrop-filter` utility to whether an element's backdrop should be rendered as sepia or in full color.

```html purple
<template preview>
  <div class="space-y-8">
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-purple-600">sepia-0</p>
      <div class="relative ml-4 h-24 w-24 rounded-xl overflow-hidden">
        <div class="absolute inset-0" style="background-image:url('https://images.unsplash.com/photo-1608501821300-4f99e58bba77?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=96&q=80&duotone=6B21A8,7E22CE&duotone-alpha=50');"></div>
        <div class="absolute inset-0 rounded-xl backdrop-filter backdrop-sepia-0 overflow-hidden"></div>
      </div>
    </div>
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-purple-600">sepia</p>
      <div class="relative ml-4 h-24 w-24 rounded-xl overflow-hidden">
        <div class="absolute inset-0" style="background-image:url('https://images.unsplash.com/photo-1608501821300-4f99e58bba77?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=96&q=80&duotone=6B21A8,7E22CE&duotone-alpha=50');"></div>
        <div class="absolute inset-0 rounded-xl backdrop-filter backdrop-sepia overflow-hidden"></div>

    </div>
  </div>
</template>

<div class="backdrop-filter **backdrop-sepia-0** ..."></div>
<div class="backdrop-filter **backdrop-sepia** ..."></div>
```

## Responsive

To control an element's backdrop sepia filter at a specific breakpoint, add a `{screen}:` prefix to any existing backdrop sepia utility. For example, use `md:backdrop-sepia-0` to apply the `backdrop-sepia-0` utility at only medium screen sizes and above.

```html
<div class="backdrop-filter backdrop-sepia **md:backdrop-sepia-0** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
