  plugin: function () {

  }
}

## Usage

Use the `sepia` and `sepia-0` utilities alongside the `filter` utility to whether an element should be rendered as sepia or in full color.

```html purple
<template preview>
<div class="space-y-8">
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-purple-600">sepia-0</p>
      <div class="ml-4 h-24 w-24 rounded-xl bg-purple-400 filter sepia-0"></div>
    </div>
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-purple-600">sepia</p>
      <div class="ml-4 h-24 w-24 rounded-xl bg-purple-400 filter sepia"></div>
    </div>
  </div>
</template>

<div class="filter **sepia-0** ..."></div>
<div class="filter **sepia** ..."></div>
```

## Responsive

To control an element's sepia filter at a specific breakpoint, add a `{screen}:` prefix to any existing sepia utility. For example, use `md:sepia-0` to apply the `sepia-0` utility at only medium screen sizes and above.

```html
<div class="filter sepia **md:sepia-0** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
