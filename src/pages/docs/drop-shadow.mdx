## Usage

Use the `drop-shadow-{amount}` utilities alongside the `filter` utility to blur an element.

```html indigo
<template preview>
  <div class="space-y-8">
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-indigo-600">drop-shadow-sm</p>
      <div class="ml-4 h-24 w-24 rounded-xl bg-white filter drop-shadow-sm"></div>
    </div>
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-indigo-600">drop-shadow</p>
      <div class="ml-4 h-24 w-24 rounded-xl bg-white filter drop-shadow"></div>
    </div>
    <div class="flex items-center">
      <p class="w-32 text-right text-sm font-mono text-indigo-600">drop-shadow-md</p>
```

## Responsive

To control an element's drop shadow at a specific breakpoint, add a `{screen}:` prefix to any existing drop-shadow utility. For example, use `md:drop-shadow-xl` to apply the `drop-shadow-xl` utility at only medium screen sizes and above.

```html
<div class="filter drop-shadow-md **md:drop-shadow-xl** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
